Get started

Up in 30 seconds.

Install via Homebrew — the same way you install everything else on your Mac.

1
Install with Homebrew
Taps the Clui cask and installs the latest signed release.
$brew install --cask Youssef2430/clui/clui
2
Launch Clui
Open it from your Applications folder or Spotlight. It will live in your menu bar.
$open -a Clui
3
Press Option+Space
That's it. The overlay appears. Your existing Claude Code session is already connected.
